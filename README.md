# Бәйшешек — тест на знание казахского языка

Приложение для проверки знания слов и выражений из учебника
**«Бәйшешек. Қазақ тілі мен әдебиеті»** для 5-классников (2-я часть, темы 7-13).

В репозитории два варианта одного и того же теста:

- **Streamlit-приложение** (`app.py`) — рекомендованное.
- **Чистый HTML+JS** (`quiz.html`) — резервный вариант, открывается двойным кликом.

Оба используют общую базу из **324 единиц** (244 слова, 49 фраз, 31 упражнение с пропусками).

---

## Streamlit-вариант (рекомендуется)

### Запуск

```bash
cd "/Users/ergalimabiev/Desktop/Новая папка"
python3 -m streamlit run app.py
```

Откроется в браузере на `http://localhost:8501`. Чтобы остановить — `Ctrl+C` в терминале.

### Что внутри

- `app.py` — приложение Streamlit
- `data.json` — база вопросов (используется `app.py`)

### Особенности

- Выбор тем (одна / несколько / все 7)
- Выбор типов заданий (kk→ru, ru→kk, фразы, пропуски)
- Размер сессии: 10 / 20 / 40 / все
- Виртуальная клавиатура с казахскими буквами (ә ө ұ ү ң қ ғ і һ) для ru→kk заданий
- Прогресс-бар и счёт «дұрыс / барлығы»
- В конце — статистика по темам, где были ошибки (поможет понять, что повторить)

---

## HTML+JS-вариант (без сервера)

### Запуск

Дважды кликнуть по `quiz.html` — откроется в браузере по умолчанию.
Не нужен Python и установка чего-либо. Работает офлайн.

### Что внутри

- `quiz.html` — приложение (HTML + CSS + JS, в одном файле)
- `data.js` — та же база вопросов, что и `data.json`, но в формате JS
- Та же виртуальная клавиатура и логика проверки

## Логика проверки ответа

- Регистронезависимая (можно писать «больница» или «БОЛЬНИЦА»)
- Лишние пробелы по краям обрезаются, повторные пробелы внутри сжимаются
- Знаки препинания и кавычки игнорируются
- Если у слова несколько переводов (например, «емхана» = больница / поликлиника), засчитывается любой из них

## Как добавить свои слова

Структура одного элемента (одинаковая в `data.json` и `data.js`):

```json
{ "theme": 7, "type": "word", "kk": "дәрігер", "ru": ["врач", "доктор"] }
```

```json
{ "theme": 8, "type": "phrase",
  "kk": "Қазақстан — менің Отаным",
  "ru": ["Казахстан — моя Родина"] }
```

```json
{ "theme": 9, "type": "fill",
  "template": "Күн — біздің ___ жұлдызымыз",
  "answer": "ең жақын",
  "hint": "ближайшая",
  "options": ["ең жақын", "ең үлкен", "ең қызық", "ең жарық"] }
```

После правки — Streamlit подхватит изменения автоматически (либо нажать «R» / «Always rerun»). HTML-вариант — обновить страницу в браузере.

Если правишь только `data.js`, не забудь синхронизировать `data.json`:

```bash
node -e "const fs=require('fs'); const w={}; eval(fs.readFileSync('data.js','utf8')); fs.writeFileSync('data.json', JSON.stringify(w.QUESTIONS, null, 2));"
```
